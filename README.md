# Bordeaux-Wine-Sales-Analysis-and-Customer-Behavior-Study

## Overview
This project analyzes Bordeaux wine sales data to explore customer purchasing behavior, sales trends, and key factors influencing customer decisions.

## Objectives
- Analyze wine sales performance
- Identify customer behavior patterns
- Discover trends in purchasing activity
- Provide business insights through data visualization

## Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Project Structure
- `data/`: raw and processed datasets
- `notebooks/`: exploratory notebooks
- `src/`: source code for analysis
- `outputs/`: generated figures and reports

## Methods
- Data cleaning
- Exploratory Data Analysis (EDA)
- Statistical analysis
- Customer behavior analysis
- Visualization

## Results
This project highlights major sales tendencies and customer behavior characteristics in the Bordeaux wine market.
